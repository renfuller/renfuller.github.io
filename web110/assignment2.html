For my first activity, I explored the "Tutorial Point: Web Browser" article. This activity explained the basic features and functions of web browsers and how they can be customized to fit different users' needs. I learned that browsers are designed to make accessing information on the internet easier by allowing users to create bookmarks, organize tabs into groups, and install extensions that add new features. The article used Internet Explorer as an example, but many of these tools are available in modern browsers as well. Overall, this activity helped me better understand how web browsers can be personalized to improve organization and make everyday internet use more efficient. For my second activity, I completed the Wikiversity hands-on exercise about installing and comparing multiple web browsers. Instead of only using the suggested examples, I explored several browsers that I was more familiar with, including Internet Explorer, Chrome, DuckDuckGo, Puffin, and Opera GX. By comparing these programs, I learned that each browser has its own strengths and unique features. Some focus on privacy, while others offer better customization, organization tools, or extension support. This activity reminded me that choosing the right browser depends on a user's personal preferences and needs.
